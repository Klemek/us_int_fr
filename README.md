# us_int_fr
Alternative US international keyboard layout to write french (make french accents reachable only with `Alt Gr`)

## Differences from US international layout

* (case sensible) `Alt Gr` + 
  * `Q` -> `Â`
  * `R` -> `Ê`
  * `Y` -> `Û`
  * `U` -> `Ù`
  * `I` -> `Î`
  * `P` -> `Ô`
  * `A` -> `À`
  * `F` -> `È`
  * `J` -> `Ï`
* Other
  * `Alt Gr` + `.` -> `·`
  * No dead keys for quote and tick
